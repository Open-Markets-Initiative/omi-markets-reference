# Omi Market Data Reference

A repository of [documents](https://github.com/Open-Markets-Initiative/omi-markets-reference/tree/main/data/documents "Omi Market Data Reference Documents") and [links](https://github.com/Open-Markets-Initiative/omi-markets-reference/tree/main/data/links "Omi Market Data Reference Links") for understanding market data, capture, storage, and analytics.

We are always on the lookout for new resources on the topics of market data, data platforms, and backtesting.  Please add any suggestions as an [issue](https://github.com/Open-Markets-Initiative/omi-markets-reference/issues "Omi Market Data Reference Issue").

## Disclaimer

Reference documents have been collected from publicly available resources.  If any information is in violation of copyright, please let us know and we will remove it.

Enjoy.
