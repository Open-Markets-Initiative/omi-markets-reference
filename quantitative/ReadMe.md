# Omi Quantitative Reference

A repository of [books](https://github.com/Open-Markets-Initiative/omi-markets-reference/tree/main/quantitative/books "Omi Quantitative Reference Books"), [documents](https://github.com/Open-Markets-Initiative/omi-markets-reference/tree/main/quantitative/documents "Omi Quantitative Reference Documents"), [links](https://github.com/Open-Markets-Initiative/omi-markets-reference/tree/main/quantitative/links "Omi Quantitative Reference Links"), and [videos](https://github.com/Open-Markets-Initiative/omi-markets-reference/tree/main/quantitative/videos "Omi Quantitative Reference Videos") for understanding quantitative finance, market microstructure, and systematic trading research.

We are always on the lookout for new resources on the topics of quantitative research, alpha modeling, execution analytics, and market microstructure.  Please add any suggestions as an [issue](https://github.com/Open-Markets-Initiative/omi-markets-reference/issues "Omi Quantitative Reference Issue").

## Disclaimer

Reference documents have been collected from publicly available resources.  If any information is in violation of copyright, please let us know and we will remove it.

Enjoy.
