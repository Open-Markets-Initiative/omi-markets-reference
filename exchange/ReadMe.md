# Omi Exchange Reference

A repository of [books](https://github.com/Open-Markets-Initiative/omi-markets-reference/tree/main/exchange/books "Omi Exchange Reference Books"), [documents](https://github.com/Open-Markets-Initiative/omi-markets-reference/tree/main/exchange/documents "Omi Exchange Reference Documents"), [links](https://github.com/Open-Markets-Initiative/omi-markets-reference/tree/main/exchange/links "Omi Exchange Reference Links"), and [videos](https://github.com/Open-Markets-Initiative/omi-markets-reference/tree/main/exchange/videos "Omi Exchange Reference Videos") for understanding the design and operation of electronic exchanges.

We are always on the lookout for new resources on electronic exchange architecture, matching engines, market structure, and exchange connectivity.  Please add any suggestions as an [issue](https://github.com/Open-Markets-Initiative/omi-markets-reference/issues "Omi Exchange Reference Issue").

## Disclaimer

Reference documents have been collected from publicly available resources.  If any information is in violation of copyright, please let us know and we will remove it.

Enjoy.
