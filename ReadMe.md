# Omi Markets Reference

A repository of [data](https://github.com/Open-Markets-Initiative/omi-markets-reference/tree/main/data "Omi Market Data Reference"), [exchange](https://github.com/Open-Markets-Initiative/omi-markets-reference/tree/main/exchange "Omi Exchange Reference"), and [quantitative](https://github.com/Open-Markets-Initiative/omi-markets-reference/tree/main/quantitative "Omi Quantitative Reference") references for understanding electronic financial markets.

We are always on the lookout for new resources on the topics of market data, electronic exchanges, and quantitative research.  Please add any suggestions as an [issue](https://github.com/Open-Markets-Initiative/omi-markets-reference/issues "Omi Markets Reference Issue").

## Open Markets Initiative

[The Open Markets Initiative](https://github.com/Open-Markets-Initiative/Directory/tree/main/About "About Omi") (Omi) is an organization dedicated to enhancing the stability of electronic financial markets using modern development methods.
